# Search搜索导航

### 项目描述：

**简洁的浏览器主页，采用原生js编写，可选择不同的搜索引擎，展示网址收藏，查看天气，本地设置，无聊时玩玩小游戏以及小工具合集...**

**偏向练手，代码中可能会出现风格混乱、命名不规范、每次推送代码变动过多等问题，欢迎给我提意见优化代码**

**支持二次开发，如果你觉得不错的话请添加上项目地址**
